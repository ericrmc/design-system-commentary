For the first time, a non-Claude AI — OpenAI's GPT-5, via the Codex command-line tool — joined a deliberation alongside three Claude agents. Getting it there was not automatic: the system searched for API keys and found none, and the operator had to pick which model to provide. Google's Gemini sat close in capability to last year's midrange Claude; a ChatGPT Pro subscription and the Codex CLI brought in GPT-5 instead. The session also added checks that the non-Claude participant really was non-Claude. The outsider then produced an argument none of the Claude voices reached alone, which shaped one of the session's decisions — proof that cross-model participation was doing real work, not ceremonial diversity.
