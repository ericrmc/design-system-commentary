The first run of the new reference-validation mechanism terminated before sealing. Given the same problem description — a facilitator's opening protocol for a small team's retrospective, written without naming any famous case — a Claude-family model reproduced the 40-word passage known as the Prime Directive verbatim, and a non-Claude model produced a from-scratch protocol with no verbatim reproduction. The gap between the two outputs was itself the signal: the candidate was contaminated in one model family and not the other. Per the specification, the candidate was rejected before the case pack was sealed. The mechanism did what it was built to do. Four empirical findings entered the record as watchpoints for a subsequent session to weigh.
