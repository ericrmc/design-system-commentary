Until Session 003, the different perspectives in each deliberation were all voiced by one AI taking turns. They could hear each other coming. Session 003 ran them as separate AI agents in parallel, in isolated contexts, so the disagreement was real instead of performed. It used this new setup to design the rules for using it. The Skeptic warned that requiring it for every decision would turn it into hollow ritual within five sessions, and the rules were tightened to require it only for genuinely consequential choices.
